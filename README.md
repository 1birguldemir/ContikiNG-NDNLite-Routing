# IoT Multi-Hop Routing Simulation 📡

This repository contains a research-focused IoT simulation project demonstrating multi-hop routing and packet structures within a network topology. 

### 🛠️ Technologies & Tools Used
* **Operating System:** Contiki-NG
* **Network Protocol:** NDN-Lite (Named Data Networking)
* **Simulator:** Cooja Simulator
* **Language:** C

### 📋 Project Overview
The main goal of this project is to simulate and test multi-hop communication in an IoT environment. It focuses on handling custom packet structures and ensuring successful data transmission across multiple nodes in a defined topology. 

### 💡 Features
* Implementation of multi-hop routing mechanisms.
* Custom packet structure design and handling.
* Simulation testing within the Cooja environment to verify routing paths and node communication.
